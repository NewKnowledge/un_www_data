New Knowledge Analysis

Using raw data from the United Nations, New Knowledge used multiple algorithmic, modeling, and visualization methodologies to present an accessible glimpse into the the progress, barriers, and supportive factors around the economic empowerment of women.

The analysis required the use of significant data visualization and augmentation methods, including machine learning based natural language processing and algorithmic geolocation. The final analysis consists of 140,855 datum over 30 features.

The analysis is presented using New Knowledge's intelligent data investigation platform with extensive use of the NK API. All visualizations automatically follow data visualization best practices and are fully interactive - with the option for static export.
